Sinusoidal Pulse Width Modulation

Advanced PWM techniques for motor control

SPWM (Sinusoidal Pulse Width Modulation) is a technique used to generate sinusoidal voltages from a DC bus using high-frequency switching.

A sinusoidal reference is compared against a high-frequency triangular carrier. Where the sine wave is higher, the inverter switch turns ON. Where it is lower, it turns OFF.

This creates a PWM waveform whose average value follows a sine wave, allowing smooth motor torque and reduced harmonics.
